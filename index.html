Predicting Supernovae

Solving the puzzle called Universe one step at a time

Supernovae is a rare astronomical event which occurs at the end of stellar evolutionary stages of a massive star's life. A supernova is typically marked by a catastrophic and dramatic explosion. Astronomers and the scientific community view supernovae as a source of information about the universe. These explosions occur billions of light years away and provide vital cues to how the universe could have been created. The detection for possible supernovae currently is done visually and taking vastness of the universe into account, a considerable chunk of supernovae go undetected. With larger telescopes capable of collecting immense amount of data, comes the need of an automatic system for classification and prediction of these events. This would not only help astronomers focus on relevant objects much quickly but also help collect more data from Supernovae. This data can be later used to corroborate theories regarding the birth of the universe. Considering the dearth of telescope resources in comparison to the universe, a system to predict would help us focus resources and collect comprehensive data. A robust visible-light and spectroscopic data based supernova detection system will be a successor to the Supernova early warning system(SNEW). The SNEW system cannot focus on other galaxies since its based off of neutrino detectors and has not issued any new supernova alerts.

Using Long Short Term Memory

Using recurrent neural network to predict noisy and irregular time series.

Structure of recurrent neural networks helps the model remember previously seen data and helps make decisions. But a drawback of this model is that after a certain number of time steps, the effect of data on the weights is completely erased and the model gives high priority to the data it has seen most recently. Long short term memory(LSTM) models work a little differently. Though the crux of the model is a recurrent neural network, the way LSTM cell is designed allows it to store information from previous time steps. The data is stored as a cell state which is updated every time a new data point is passed through. This bypasses the vanishing gradient problem which plagues recurrent neural networks. This allows for a possibility of predicting difficult time series using these models. The large number of possibilities in terms of the number of architectures which could effectively work for the current problem. In addition to that, it lays the foundation for adapting machine learning to more obscure fields to solve basic problems.

Predicting the Long Period Variables

Using LSTM models to predict Long period variables

Training LSTM on entire LPV dataset

The LSTM sees the entire LPV dataset and with a cell state size of 200 and then makes a prediction on the next point.This was to assess if the LSTM could predict the LPV to any degree.

Training LSTM on training LPV dataset

The LSTM is trained on half the LPV dataset and with a cell state size of 50 and then makes a prediction on the next point. This test was carried out to assess model robustness.

Training LSTM on 20% training LPV dataset

The LSTM sees only 20% of the training dataset. The cell state size of 50 and then makes a prediction on the next point. This test was to assess the least data size to get accurate predicitons from the model.
