Irregular Time Series Prediction

Using Long Short Term Memory Models for Prediction

Structure of recurrent neural networks helps the model remember previously seen data and helps make decisions. But a drawback of this model is that after a certain number of time steps, the effect of data on the weights is completely erased and the model gives high priority to the data it has seen most recently. Long short term memory(LSTM) models work a little differently. Though the crux of the model is a recurrent neural network, the way LSTM cell is designed allows it to store information from previous time steps. The data is stored as a cell state which is updated every time a new data point is passed through. This bypasses the vanishing gradient problem which plagues recurrent neural networks. This allows for a possibility of predicting difficult time series using these models. The large number of possibilities in terms of the number of architectures which could effectively work for the current problem. In addition to that, it lays the foundation for adapting machine learning to more obscure fields to solve basic problems.

The amount of data which is remembered by this model is dependent on the size of the cell state which must be chosen clearly. The data we intend to use to train the networks have a small number of points. LSTM requires a model for every object individually. We intend to look for a way we could train a central set of additional weights for the model using all the data available for supernovae. This would prevent the training to be localized for a single object. This should make the model more robust in detecting and predicting supernovae since the model will now have more to work off of. We still need to inspect if the data should be normalized considering that every star has a different brightness baseline. Hence this problem involves challenges which will force us to look at the LSTM structure more precisely.

Testing with Sythentic data

Astronomical data can be abruptly noisy which makes it a hard problem for modern machine learning methods to tackle. Moving from Supernovae to other stars, prediction problems lie for star types like Long period variables and to predict astronomical phenomenon like Micro-lensing. The data is not only noisy but also irregularly spaced. Before implementing the LSTM methods on the astronomical data, it was warranted to test on sythentic data. The synthetic data we use for testing are sinusoidal curves, gaussian processes and impulses. The impulse data serves to simulate supernovae with different light curve gradients.

Training and Evaluation of LSTM's

With the fast growth in machine learnign research, there have been many new methods to train neural networks. We intended on going step by step up the complexity ladder only if the existing method failed. We tried the vanilla stochastic gradient descent, nesterov's momentum and adadelta(adaptive learning rate algorithm) to train the models. As you can see below, the adadelta method provides stable training and usually converges on a practical solution.

Models are evaluated on the basis of the root mean squared error(RMSE). Its a well accepted method of comparing forecasting errors of different models for a particular variable. he error is key in determining the efficiency of training. It also helps prevent overfitting and can benchmark the instant after which the model starts learning on the data noise.
